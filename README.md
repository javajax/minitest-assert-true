# Minitest::AssertTrue

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'minitest/assert_true'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install minitest-assert-true

## Usage

require 'minitest/assert_true'
assert_true true, "this should work"
